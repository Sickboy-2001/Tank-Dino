# Tank-Dino
A re-vamped version of Google's popular Dino game!


The world is changing. No longer must Dinos be left to their own devices to traverse the harsh pre-historic terrain. In this re-vamped version of Google’s popular Dino game, our beloved protagonist has at his disposal none other than a standard-issue M1 Abrams tank, courtesy of the American taxpayer. See how far you can get!

Give it a try here: https://sickboy-2001.github.io/Tank-Dino/
